# Monash Waste Helper

A small mobile-first PWA for checking which City of Monash household bin an item belongs in.

## Use

Open `index.html` in a browser, or host the folder on GitHub Pages / Netlify / Cloudflare Pages.

For iPhone, open the hosted URL in Safari and choose **Share > Add to Home Screen**.

## Files

- `index.html` - app shell
- `styles.css` - visual styles
- `app.js` - searchable waste item data and app logic
- `manifest.webmanifest` - PWA manifest
- `sw.js` - offline cache service worker
- `icon.svg` - app icon

## Source note

The waste rules were manually summarised from City of Monash waste and recycling pages and should be checked against the official website if council guidance changes.
